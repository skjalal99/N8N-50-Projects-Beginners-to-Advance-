# Google Forms → Google Sheets Lead Collector

## Overview

An n8n automation workflow that captures leads from Google Forms and automatically stores them in Google Sheets.

## Features

- Automatic lead collection
- Data formatting
- Email validation
- Google Sheets storage
- Confirmation emails

## Workflow

Google Form → n8n → Google Sheets → Gmail

## Tools Used

- n8n
- Google Forms
- Google Sheets
- Gmail API

## Setup

1. Connect Google credentials in n8n
2. Import workflow JSON
3. Configure Google Sheet
4. Activate workflow

## Use Cases

- Marketing agencies
- Freelancers
- SaaS companies
- Sales teams
