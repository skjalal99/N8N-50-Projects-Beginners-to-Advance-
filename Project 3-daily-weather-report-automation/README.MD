# Daily Weather Report Automation 🌤️

An n8n workflow that automatically collects daily weather data and sends a personalized weather report.

## Features

- Automated daily execution
- Weather API integration
- Email notifications
- Weather history tracking

## Workflow

Schedule Trigger
→ Weather API
→ Data Processing
→ Email Report

## Tools Used

- n8n
- OpenWeather API
- Gmail API
- Google Sheets

## Setup

1. Create API credentials
2. Import n8n workflow
3. Add environment variables
4. Activate workflow
